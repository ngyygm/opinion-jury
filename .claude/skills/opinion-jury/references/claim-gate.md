# CLAIM_GATE

Run before hearings. Extract verifiable claims, query authoritative sources when available, save evidence ledgers, and classify each claim:

```text
VERIFIED | UNVERIFIED | CONTRADICTED | CATEGORY_MISMATCH |
OPINION_PRESENTED_AS_FACT | NOT_APPLICABLE
```

Do not treat simulated actor speech as verified evidence. Actors may receive limited observable subsets rather than the full packet when realism requires incomplete information. The blind adjudicator and final replay analyst receive the full neutral packet.
